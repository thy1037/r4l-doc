寄存器说明

MAC

// 非零寄存器
Reg0 : 0800a003
Reg40: 000000aa
Reg42: 03020100
Reg44: 00040008
Reg53: 03e80000
  • Reg 0
    0b0000_1000_0000_0000_xx10_0000_0000_0011
    使能校验、发送、接收，
    其中xx为速度，包括后面的1是全双工模式，都是需要和 PHY 联合设置的。
bit 15bit 14速度
1010Mbps
11100Mbps
001Gbps
012.5Gbps

  • Reg 40
    可以固定，值为 0x000000AA
    使能接收队列0、1、2、3,并用于DCB/Generic

  • Reg 42
    可以固定，值为 0x03020100
    设置接收队列0～3的优先级分别为3～0

  • Reg 44
    中断状态寄存器，不可写入固定值
    当前 0x00040008 表示：MDIO 中断、PHY 中断

  • Reg 45
    可以固定，值为 0x03e80000
    一个定时器，当前值为 1000ms

DMA CTRL

// 非零寄存器
Reg0: 00010000
Reg1: 0f0f080f
Reg3: 03030300
Reg4: 00000003

  • Reg 0
    可以固定，值为 0x00010000
    表示传送完成可以触发中断，与 ETH_DMACIER 联合工作。

  • Reg 1
    可以固定，值为 0x0f0f080f
    表示AXI 最大未完成写请求数为16、读请求数为16；使能增强地址模式；固定突发长度，长度可以是1、4、8、16

  • Reg 3 Reg 4
    可以固定，值为 0x00000000
    调试寄存器，可以不写，保持默认值

DMA CH0

// 非零寄存器
Reg0 : 00010000
Reg1 : 00080011
Reg2 : 00080c01
Reg5 : 83194000
Reg7 : 83186000
Reg8 : 83194000
Reg10: 83188000
Reg11: 000001ff
Reg12: 000001ff
Reg13: 0000d041
Reg15: 000007c0
Reg17: 83194000
Reg19: 83186000
Reg22: 00000001
Reg23: 99a5d040

  • Reg 0
    可以固定，值为 0x00010000
    表示使能 8xPBL 模式

  • Reg 1
    可以固定，值为 0x00080011，写入后开启发送
    表示设置传输突发长度、使能提前处理第二个数据包、开启传输

  • Reg 2
    可以固定，值为 0x00080c01，写入后开启接收
    表示设置接收突发长度、接收缓冲区长度、开启接收

  • Reg 5
    不可以固定
    表示发送描述符列表的起始地址

  • Reg 7
    不可以固定
    表示接收描述符列表的起始地址

  • Reg 8
    不可以固定
    表示发送描述符列表的结束地址

  • Reg 10
    不可以固定
    表示接收描述符列表的结束地址

  • Reg 11
    可以固定，值为 0x000001ff
    表示环形发送描述符长度，此处为512

  • Reg 12
    可以固定，值为 0x000001ff
    表示环形接收描述符长度，此处为512

  • Reg 13
    可以固定，值为 0x0000d041
    中断使能，表示使能发送、Tx缓冲区满、接收、提前接收中断，发送停止、Rx缓冲区满、接收停止、接收看门狗、提前发送、致命错误中断。

  • Reg 15
    可以固定，值为 0x000007c0
    此处值为复位默认值，可以不用写，表示的是 TxDMA 工作间隔时间 125us

  • Reg 17
    只读寄存器
    表示当前发送描述符

  • Reg 19
    只读寄存器
    表示当前接收描述符

  • Reg 22
    只读寄存器`
    当前接收缓冲区地址指针的高8位

  • Reg 23
    只读寄存器
    当前接收缓冲区地址指针的低32位
